## Usage
### Environment
```bash
conda create -n py311 python=3.11 -y
conda activate py311
pip install -r requirements.txt
```

### Download
```bash
git clone git@github.com:MuGemSt/DyberGenshinAI-VITS.git
cd DyberGenshinAI-VITS
```

### Run
```bash
python main.py
```
